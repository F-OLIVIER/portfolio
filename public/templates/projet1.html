DEMANDE CLIENT

Crée une application pour simplifier et gagner du temps lors de la préparation des combats de Guildes pour le jeu Conqueror's Blade

SOLUTION PROPOSÉ

Un système de gestion multi-plateforme composé de 3 applications ayant une base de données commune :
• Un bot Discord permettant de collecter les inscriptions des utilisateurs
• Un site internet avec deux types d'accès dépendant du rôle Discord
• Une application mobile permettant au utilisateur de renseigner leurs informations

Le bot Discord ainsi que le site internet disposent de deux types d'accès :
• Un accès membre permettant aux utilisateurs de renseigner leurs informations.
• Un accès gestionnaire permettant aux préparateurs d'administrer les informations demandées aux utilisateurs ainsi que de créer et d'optimiser les groupes avec les informations renseignées par les utilisateurs.

Deux langues sont actuellement prisent en charge, le Français et l'anglais.
Le système est déployé sur un serveur PulseHeberg avec un nom de domaine associé.
Voici la page du site internet qui explique le projet aux potentiels utilisateurs : https://easymanagementtw.fr/description
