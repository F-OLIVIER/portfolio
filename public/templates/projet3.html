DEMANDE CLIENT

Ce projet a été réalisé dans le cadre de ma formation. Le but a été de recréer un réseau social tels que Facebook ou Linkedin.

SOLUTION PROPOSÉ

Le projet est réalisé en Next.js et intègre les fonctionnalités suivantes :

  • Un système de migration de la base de données SQL
  • Création est gestion de post
  • Gestion de groupes de discussion
  • Gestion de suivi (follow)
  • Gestion d'affichage des post en fonction du type de profil
  • Gestion d'événement via des groupes
  • Gestion des notifications (en ligne et hors ligne)
